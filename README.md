Madrona Ridge: Sovereign Pentad Archive Located on Mudge Island, BC, Madrona Ridge is a heroic vineyard station focused on the Sovereign Pentad (Coronation, Opal, Rose, Noir, Tiara). By trialing these hybrids on 200ft volcanic terraces, the project preserves BC’s heritage and studies how mineral-rich, maritime soils influence phenolic development.
